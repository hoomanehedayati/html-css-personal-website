# Statement of Purpose (SOP)

This folder contains the Statement of Purpose document for university applications.

## File

- `Statement-of-Purpose.md` - Complete SOP document in Markdown format

## Usage

1. Review the SOP document
2. Customize the "Why This Program" section for each university you apply to
3. Convert to PDF format for submission
4. Adjust word count if required by specific programs

## Tips

- Most programs require 1000-1500 words
- Customize for each university/program
- Highlight specific faculty or research areas
- Show genuine interest and alignment with program goals
- Proofread carefully before submission

## Converting to PDF

You can convert the Markdown file to PDF using:
- Online Markdown to PDF converters
- Pandoc: `pandoc Statement-of-Purpose.md -o Statement-of-Purpose.pdf`
- VS Code with Markdown PDF extension
